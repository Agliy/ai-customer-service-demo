API 接口列表

POST /api/chat
在线

AI 聊天接口，接收用户消息并返回 AI 回复内容。支持多轮对话上下文。

JSON 
{
  "session_id": "sess_abc123",
  "message": "这款手机的续航怎么样？",
  "context": [
    { "role": "user", "content": "推荐一款手机" },
    { "role": "assistant", "content": "推荐 iPhone 15 Pro" }
  ]
}
JSON 
{
  "code": 200,
  "data": {
    "reply": "iPhone 15 Pro 视频播放最长可达 23 小时，支持 20W 快充。",
    "session_id": "sess_abc123",
    "message_id": "msg_789xyz"
  }
}
GET /api/knowledge
在线

知识库查询接口，根据关键词检索知识库中的问答对及文档片段。

Query Params 
GET /api/knowledge?q=手机续航&limit=5&category=数码
JSON 
{
  "code": 200,
  "data": {
    "total": 12,
    "results": [
      {
        "id": "kb_001",
        "question": "iPhone 15 Pro 续航多久？",
        "answer": "视频播放最长 23 小时。",
        "score": 0.96
      }
    ]
  }
}
POST /api/brand
在线

品牌信息更新接口，用于动态更新客服助手的品牌介绍、回复风格等配置。

JSON 
{
  "brand_name": "潮品数码商城",
  "description": "专注高端数码产品，正品保障，7天无理由退货。",
  "tone": "friendly",
  "keywords": ["正品", "快充", "续航", "售后"]
}
JSON 
{
  "code": 200,
  "data": {
    "updated": true,
    "brand_id": "brand_001",
    "updated_at": "2026-06-28T14:30:00Z"
  }
}
GET /api/statistics
在线

统计数据查询接口，返回客服系统的会话量、响应率、用户满意度等核心指标。

Query Params 
GET /api/statistics?start=2026-06-01&end=2026-06-28
JSON 
{
  "code": 200,
  "data": {
    "sessions": 3428,
    "messages": 12856,
    "avg_response_time": 420,
    "satisfaction_rate": 0.94,
    "resolution_rate": 0.87
  }
}
GET /api/logs
在线

操作日志查询接口，分页返回系统操作日志，支持按级别、时间范围筛选。

Query Params 
GET /api/logs?level=info&page=1&pageSize=20
JSON 
{
  "code": 200,
  "data": {
    "total": 156,
    "page": 1,
    "pageSize": 20,
    "logs": [
      {

        "level": "info",
        "message": "用户会话创建成功",
        "source": "chat-service"
      }
    ]
  }
}
